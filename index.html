This is my second project in DecodeLabs Internship with responsive frontend web layout built using HTML and CSS with a mobile-first design approach in Decode. It features a modern navigation bar with hamburger menu behavior, a functional sidebar, a hero section with image and content alignment, and a structured footer. The project demonstrates the use of CSS Grid for macro layout, Flexbox for component alignment, responsive media queries, fluid sizing, and accessible user interface design optimized for mobile, tablet, and desktop screens.
